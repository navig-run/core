# Support

NAVIG is a solo-maintained project.

## Questions and usage help

Use [GitHub Discussions](https://github.com/navig-run/core/discussions) for:
- Usage questions
- Command help
- Integration ideas
- Anything that isn't a reproducible bug

## Bug reports

Open an [Issue](https://github.com/navig-run/core/issues/new/choose) using the bug report template.
Include your NAVIG version (`navig --version`), OS, and exact command output.

## Security vulnerabilities

Do **not** open a public issue. Follow the process in [SECURITY.md](SECURITY.md):
private advisory via GitHub → email fallback at `security@navig.run`.

## Response time

No SLA guarantee. This is a solo project. Best-effort response within a few days.
