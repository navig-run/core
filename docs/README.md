# Public scripts

- `bootstrap_navig_linux.sh` — first-time Ubuntu bootstrap for NAVIG hosts.
- `github-bootstrap.sh` — repository/community bootstrap helper.
- `install_navig_factory_server.sh` — install the operational factory server as a service.
- `install_navig_linux.sh` — standard Linux installer.
- `install_navig_linux_enhanced.sh` — enhanced Linux installer.
- `install_navig_macos.sh` — macOS installer.
- `install_navig_windows.ps1` — standard Windows installer.
- `install_navig_windows_enhanced.ps1` — enhanced Windows installer.
- `install_matrix_synapse_windows.cmd` — Windows entrypoint for Matrix/Synapse install.
- `install_matrix_synapse_windows.ps1` — PowerShell Matrix/Synapse install script.
- `navig_quick_setup.ps1` — quick Windows setup helper.
- `navig_quick_setup.sh` — quick Unix setup helper.
- `navig-tunnel.service` — systemd unit for the NAVIG tunnel service.
- `navig-tunnel.sh` — tunnel launcher script.
- `bridge-tunnel.service` — systemd unit for the bridge tunnel service.
