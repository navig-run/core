# Brand Usage Guidelines

## Protected Names and Assets

The following names and brand elements are reserved for the official NAVIG project:
- **NAVIG** (product name)
- **SCHEMA** (organization name)
- NAVIG logos and visual brand assets
- The 🦑 Kraken symbol as used in project code name NAVIG branding

## What You CAN Do

✅ **Fork and modify** NAVIG under the MPL-2.0 license  
✅ **Reference NAVIG** with phrases like "fork of NAVIG" or "based on NAVIG"  
✅ **Use in documentation** and technical tutorials  
✅ **Study and modify** the source code freely

## What You CANNOT Do (Without Permission)

❌ **Distribute a fork** using the "NAVIG" or "SCHEMA" name  
❌ **Use logos or brand assets** in derivative works  
❌ **Claim official status** for your fork  
❌ **Imply endorsement** by SCHEMA or NAVIG maintainers

## If You Redistribute a Modified Version

When distributing a modified version of NAVIG, you must:

1. **Rename the product** to something other than "NAVIG"
2. **Remove or replace** all NAVIG/SCHEMA branding assets (logos, icons, graphics)
3. **Retain copyright notices** in MPL-2.0 covered files (as required by the license)
4. **Include clear attribution**: "Based on NAVIG by SCHEMA"

### Example Attribution

```
MyTool (formerly based on NAVIG by SCHEMA)
Licensed under MPL-2.0
https://github.com/navigrun/navig-core
```
