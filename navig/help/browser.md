# browser

Browser automation for web tasks.

Common commands:
- `navig browser open <url>` — navigate to URL
- `navig browser click <selector>` — click element
- `navig browser fill <selector> <value>` — fill form field
- `navig browser screenshot` — capture page screenshot
- `navig browser status` — show browser status
- `navig browser stop` — stop browser

Examples:
- `navig browser open https://example.com`
- `navig browser screenshot --output page.png`
