# ahk

AutoHotkey v2 automation (Windows only).

Common commands:
- `navig ahk install` — detect or install AHKv2
- `navig ahk status` — show AHK status
- `navig ahk run <script>` — execute AHK script file
- `navig ahk exec "code"` — execute inline AHK code
- `navig ahk click x y` — click at screen coordinates
- `navig ahk type "text"` — type text with keyboard
- `navig ahk windows` — list all visible windows

Examples:
- `navig ahk run myscript.ahk`
- `navig ahk type "Hello World"`
- `navig ahk click 500 300`
