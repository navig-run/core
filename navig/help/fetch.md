# fetch

Fetch and extract content from URLs.

Usage:
- `navig fetch <url>` — fetch content from URL
- `navig fetch <url> --mode markdown` — extract as markdown
- `navig fetch <url> --mode text` — extract as plain text
- `navig fetch <url> --json` — output in JSON format

Examples:
- `navig fetch https://example.com`
- `navig fetch https://docs.python.org/3/ --mode markdown`
