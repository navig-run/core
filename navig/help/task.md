# task

Task queue for async operations.

Common commands:
- `navig task list` — list queued tasks
- `navig task add` — add a task to queue
- `navig task show <name>` — show task details
- `navig task run <name>` — run a task
- `navig task remove <name>` — remove a task

Examples:
- `navig task list --json`
- `navig task run deploy`
