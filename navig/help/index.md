# NAVIG Help

Use `navig <cmd> --help` for full CLI help, or `navig help <topic>` for a short, predictable summary.

**Infrastructure:**
- `host` — remote server connections
- `local` — local machine diagnostics
- `hosts` — /etc/hosts file management
- `tunnel` — SSH tunnels

**Services:**
- `app` — application management
- `docker` — containers
- `web` — webserver operations
- `gateway` — agent gateway server
- `service` — persistent daemon management

**Data:**
- `db` — database operations
- `file` — file upload/download/edit
- `log` — remote log files
- `backup` — backup and restore

**Automation:**
- `flow` — workflows (canonical; `task` is an alias)
- `cron` — job scheduling
- `trigger` — event-driven automation
- `scaffold` — project templates
- `skills` — AI skill definitions

**AI & Intelligence:**
- `ai` — AI assistant
- `ai-providers` — provider configuration
- `memory` — conversation memory
- `suggest` — smart suggestions
- `insights` — operations analytics

**Tools:**
- `config` — NAVIG settings
- `context` — host/app context
- `history` — command history
- `mcp` — MCP server integrations
- `wiki` — knowledge base
- `pack` — operations bundles

**Utilities:**
- `search` — web search
- `fetch` — URL content extraction
- `docs` — documentation search
- `version` — version info

**Agent & Autonomous:**
- `agent` — autonomous agent
- `heartbeat` — health checks
- `approve` — approval system
- `browser` — browser automation
- `ahk` — AutoHotkey (Windows)
- `calendar` — calendar events
- `email` — email operations

Examples:
- `navig help db`
- `navig help file`
- `navig db --help`
