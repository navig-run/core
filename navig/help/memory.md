# memory

Conversation memory and knowledge base for AI sessions.

Common commands:
- `navig memory sessions` — list conversation sessions
- `navig memory history <session>` — show session messages
- `navig memory clear` — clear session or all memory
- `navig memory knowledge` — manage knowledge entries
- `navig memory stats` — show memory statistics
- `navig memory bank` — memory bank status
- `navig memory search <query>` — search memory bank

Examples:
- `navig memory sessions --json`
- `navig memory stats`
- `navig memory search "nginx config"`
