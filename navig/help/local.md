# local

Local machine operations and diagnostics.

Common commands:
- `navig local show` — show system information
- `navig local audit` — security audit of local machine
- `navig local ports` — list open ports
- `navig local firewall` — show firewall status
- `navig local ping <host>` — ping remote host
- `navig local dns <domain>` — DNS lookup
- `navig local interfaces` — show network interfaces

Examples:
- `navig local show --json`
- `navig local ports`
- `navig local dns example.com`
